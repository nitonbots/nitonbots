nitonbots

a Coinbase(d) trading bot.

Install Niton LTC - (download installer)
Navigate to the installer and run these commands:
sudo chmod u+x nitonltc
sudo ./nitonltc install

Install Niton BTC - (download installer)
Navigate to the installer and run these commands:
sudo chmod u+x nitonbtc
sudo ./nitonbtc install
